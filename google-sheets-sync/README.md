# Google Sheets Sync — אבי ורדי הסעות

סקריפט זה מסנכרן את פניות הלקוחות מ-Supabase ל-Google Sheets אוטומטית כל יום בשעה 08:00.

## הגדרה ראשונית (פעם אחת בלבד)

1. פתחו [Google Sheets](https://sheets.google.com) וצרו גיליון חדש.
2. לחצו על **Extensions → Apps Script**.
3. מחקו את כל הקוד הקיים בעורך.
4. הדביקו את תוכן הקובץ `SupabaseSync.gs`.
5. לחצו **Save** (Ctrl+S).
6. בתפריט הנפתח של הפונקציות, בחרו **`setupTrigger`**.
7. לחצו על ▶ **Run** ואשרו את ההרשאות שהדפדפן יבקש.

לאחר ההגדרה, הגיליון יתעדכן אוטומטית כל יום בשעה 08:00 (שעון ישראל).

## הרצה ידנית

בכל עת ניתן לבחור את הפונקציה **`syncLeads`** וללחוץ ▶ Run לעדכון מיידי.

## מבנה הגיליון

| עמודה | תוכן |
|-------|------|
| # | מספר שורה |
| שם | שם הלקוח |
| טלפון | מספר הטלפון |
| תאריך נסיעה | תאריך מבוקש |
| הערות | פרטים נוספים |
| נוצר ב | תאריך ושעת הפנייה (שעון ישראל) |

## ביטול העדכון האוטומטי

הריצו את הפונקציה **`removeTrigger`** כדי לבטל את ה-Trigger.
